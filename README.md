# kitchen-riley-game10020-a2
Unity Version: 2022.3.62f2

#### Purpose: This game repository is use to show the basic design of a top-down dungeon game with a event system design for Assignment 2: Event-Driven Systems Implementation.

The gameplay below shows emergent gameplay that has a different design and a different system within the game using event functions. Fundamentally you explore around the area and collect coins to pick up and use it for points for later events or levels, you can also press the switch button onces you collect all the coins which can lower down the wall and you can press the button again to rasied up the wall. You can also collect a key (which is a cube) that will trigger an event that the door is open and you can walk inside the room area. These events are rasied and any object that is in the area can be interact with responds to events with their behviours that are being called/trigger in the game.
